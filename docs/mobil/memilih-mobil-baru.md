# Cara Memilih Mobil Baru

Panduan step-by-step sebelum membeli mobil baru agar tidak menyesal.

## 1. Tentukan Kebutuhan

| Kebutuhan | Rekomendasi Tipe |
|-----------|-----------------|
| Harian dalam kota | City car / LCGC |
| Keluarga kecil (≤4 org) | SUV compact / sedan |
| Keluarga besar (≥5 org) | MPV / SUV 7-seater |
| Off-road / medan berat | SUV 4WD / double cabin |
| Niaga / angkut barang | Pick-up / van |

## 2. Tentukan Budget

```
Harga mobil     : Rp X
DP (20–30%)     : Rp X × 20%
Cicilan/bulan   : (Harga - DP) ÷ tenor (bulan) + bunga
Biaya tambahan  : Asuransi + pajak + aksesori
```

**Patokan cicilan aman**: Maksimal 30% dari penghasilan bulanan.

## 3. Pilih Transmisi

| Tipe | Keunggulan | Kekurangan |
|------|-----------|------------|
| Manual | Lebih irit BBM, murah servis | Butuh skill, capek di macet |
| Matic (AT/CVT) | Mudah dikemudikan | Lebih boros, servis lebih mahal |

## 4. Merek Terpercaya di Indonesia

| Merek | Model Populer | Kelebihan |
|-------|--------------|-----------|
| Toyota | Avanza, Raize, Rush | Nilai jual kembali tinggi, bengkel luas |
| Honda | Brio, HR-V, CR-V | Teknologi modern, irit BBM |
| Suzuki | Ertiga, Jimny | Harga terjangkau, tangguh |
| Daihatsu | Ayla, Sigra, Rocky | LCGC terbaik, harga murah |
| Mitsubishi | Xpander, Pajero | MPV dan SUV terbaik |

## 5. Checklist Sebelum Test Drive

- [ ] Cek dimensi sesuai kebutuhan keluarga
- [ ] Pastikan ground clearance cukup untuk kondisi jalan lokal
- [ ] Tanyakan fitur keselamatan (airbag, ABS, traction control)
- [ ] Simulasi kredit di website resmi pabrikan
- [ ] Bandingkan minimal 2–3 model sebelum memutuskan

## 6. Dokumen Saat Pembelian

- KTP + KK (untuk kredit)
- NPWP (jika ada, untuk kemudahan proses)
- Slip gaji 3 bulan terakhir (untuk KPM/kredit)
- Rekening koran 3 bulan (untuk kredit)

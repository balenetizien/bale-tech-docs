# Cara Memilih Motor Sesuai Kebutuhan

## Tipe Motor dan Kegunaannya

| Tipe | Kegunaan Ideal | Contoh Model |
|------|---------------|-------------|
| Matic (skuter) | Harian dalam kota, kemacetan | Honda Beat, Yamaha Mio |
| Matic premium | Harian + perjalanan jauh | Honda Vario, Yamaha NMAX |
| Bebek (cub) | Harian irit, jarak jauh | Honda Revo, Yamaha Vega |
| Sport 150cc | Sporty + harian | Yamaha R15, Honda CBR150R |
| Trail/adventure | Off-road + touring | Honda CRF, Yamaha WR155 |

## Pertimbangan Utama

### Kapasitas Mesin

| CC | Penggunaan | Konsumsi BBM |
|----|------------|-------------|
| 100–125cc | Kota, jarak dekat | Sangat irit (55–70 km/L) |
| 150–160cc | Kota + luar kota | Irit (40–55 km/L) |
| 250cc+ | Touring, highway | Standar (25–35 km/L) |

### Transmisi

| Tipe | Cocok Untuk | Harga |
|------|------------|-------|
| Matic CVT | Pemula, perempuan, kemacetan | Lebih mahal |
| Manual | Pria dewasa, jarak jauh, off-road | Lebih murah |

## Checklist Sebelum Beli Motor

- [ ] Sesuaikan tinggi motor dengan tinggi badan
- [ ] Test ride minimal 10 menit di berbagai kondisi jalan
- [ ] Cek konsumsi BBM real di forum/review
- [ ] Tanyakan biaya servis berkala resmi
- [ ] Bandingkan garansi antar merek
- [ ] Cek ketersediaan suku cadang di kotamu

# Jadwal Servis Motor

## Tabel Interval Perawatan Motor

| Komponen | Matic (km) | Manual (km) | Keterangan |
|----------|-----------|------------|------------|
| Oli mesin | 2.000–4.000 | 2.000–4.000 | Lebih cepat jika sering kemacetan |
| Filter oli | 8.000 | 8.000 | Ganti bersamaan servis besar |
| Busi | 8.000–16.000 | 8.000–16.000 | Cek warna elektroda |
| Filter udara | 8.000–16.000 | 8.000–16.000 | Bersihkan setiap 4.000 km |
| Rantai (manual) | — | Setiap 2.000 km | Lumasi, stel kekencangan |
| V-belt / CVT belt (matic) | 20.000–24.000 | — | Wajib ganti sebelum putus |
| Roller CVT (matic) | 20.000–24.000 | — | Cek jika akselerasi melemah |
| Kampas rem depan | 15.000–20.000 | 15.000–20.000 | Cek ketebalan |
| Kampas rem belakang | 20.000–25.000 | 20.000–25.000 | Cek ketebalan |
| Ban | 15.000–20.000 | 15.000–20.000 | Atau jika sudah gundul |

## Checklist Servis Ringan Motor (2.000 km)

- [ ] Ganti oli mesin
- [ ] Cek tekanan ban
- [ ] Cek kondisi rem (pakem?)
- [ ] Cek lampu dan klakson
- [ ] Periksa rantai / tegangan CVT

## Checklist Servis Besar Motor (8.000 km)

- [ ] Semua poin servis ringan
- [ ] Ganti / bersihkan filter udara
- [ ] Ganti busi (jika perlu)
- [ ] Periksa karburator / throttle body
- [ ] Cek kampas rem
- [ ] Stel klep (setiap 16.000 km)
- [ ] Cek kondisi aki

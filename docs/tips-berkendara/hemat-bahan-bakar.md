# Cara Hemat Bahan Bakar

## Faktor yang Mempengaruhi Konsumsi BBM

| Faktor | Pengaruh ke Konsumsi BBM |
|--------|------------------------|
| Kecepatan | Makin cepat, makin boros (terutama >100 km/j) |
| Akselerasi | Akselerasi mendadak boros 20–30% lebih banyak |
| Tekanan ban | Ban kurang angin: boros 3–5% |
| AC | Menyalakan AC: boros 10–15% |
| Beban | +100 kg beban: boros 5–7% |
| Kondisi mesin | Mesin tidak dirawat: boros 15–25% |

## Tips Hemat BBM Mobil

### Saat Berkendara
1. **Akselerasi halus** — injak gas perlahan, jangan hentakan
2. **Kecepatan konstan** — ideal 60–80 km/j di jalan biasa
3. **Engine brake** — angkat gas lebih awal sebelum deselerasi
4. **Matikan mesin** jika parkir > 3 menit
5. **Gunakan cruise control** di tol (jika ada)

### Perawatan yang Hemat BBM
- Ganti busi tepat waktu (busi aus = pembakaran tidak sempurna)
- Filter udara bersih = campuran udara:bensin lebih efisien
- Tekanan ban sesuai rekomendasi pabrik
- Servis rutin tepat waktu

## Tips Hemat BBM Motor

```
Tekanan ban ideal:
- Depan : 28–30 psi (tanpa bonceng) / 30–32 psi (bonceng)
- Belakang: 30–33 psi (tanpa bonceng) / 32–36 psi (bonceng)
```

| Kebiasaan Buruk | Pemborosan BBM |
|----------------|----------------|
| Tarik gas mendadak | +15–25% boros |
| Ban kurang angin | +3–5% boros |
| Mesin dingin langsung gas | +10% boros |
| Filter udara kotor | +5–10% boros |
| Rantai terlalu kencang/kendur | +3–5% boros |
